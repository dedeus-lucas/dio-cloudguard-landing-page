# 🚀 DIO Challenge — Landing Page with HTML & CSS

## 🇧🇷 Versão em Português

## 📌 Sobre o Projeto

Este projeto foi desenvolvido como parte de um desafio educacional da plataforma **Digital Innovation One (DIO)**.

O objetivo do desafio foi criar uma **Landing Page institucional utilizando apenas HTML e CSS**, aplicando conceitos fundamentais de desenvolvimento web como:

- Estrutura semântica com HTML5  
- Estilização moderna com CSS3  
- Organização de projeto  
- Layout responsivo  
- Boas práticas de desenvolvimento frontend  

Apesar da simplicidade das tecnologias utilizadas, a proposta deste projeto foi **ir além do básico**, explorando uma abordagem mais profissional de organização de código e design de interface.

O projeto apresenta a landing page de um conceito fictício chamado **CloudGuard AI**, uma plataforma imaginária voltada para **monitoramento e segurança de infraestruturas em cloud utilizando inteligência artificial**.

A ideia foi demonstrar que **mesmo com tecnologias fundamentais como HTML e CSS é possível construir interfaces modernas, organizadas e visualmente agradáveis**.

---

## 🎯 Objetivos do Projeto

- Demonstrar domínio dos fundamentos de **HTML5 e CSS3**
- Criar uma **estrutura semântica bem organizada**
- Desenvolver uma **landing page clara e moderna**
- Aplicar **boas práticas de organização de repositório**
- Simular um projeto com padrão mais próximo de um ambiente profissional

---

## 🧱 Tecnologias Utilizadas

- HTML5  
- CSS3  

Nenhum framework ou biblioteca externa foi utilizado.  
Todo o layout e comportamento visual foram construídos **exclusivamente com HTML e CSS**.

# 🇺🇸 English Version

## 📌 About the Project

This project was developed as part of an educational challenge from the **Digital Innovation One (DIO)** platform.

The purpose of the challenge was to build an **institutional landing page using only HTML and CSS**, applying fundamental web development concepts such as:

- Semantic structure with HTML5  
- Modern styling with CSS3  
- Project organization  
- Responsive layout  
- Frontend development best practices  

Although the technologies used are simple, the intention of this project was to **go beyond the basics**, exploring a more professional approach to code organization and interface design.

The landing page presents a fictional concept called **CloudGuard AI**, an imagined platform focused on **monitoring and securing cloud infrastructures using artificial intelligence**.

The goal was to demonstrate that **even with foundational technologies like HTML and CSS it is possible to build modern, organized and visually appealing interfaces**.

- CSS3
